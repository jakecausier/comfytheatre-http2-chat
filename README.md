# HTTP/2 Chat Example

HTTP/2 Chat Example shows how to implement a bi-directional connectivity over HTTP/2.
This approach can be considered as replacement for web-sockets.

## Requirements
- Node.js >= 9.8.0

## Run

```sh
npm install
npm start
```

Open: https://localhost:4000
